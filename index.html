About Code Wars


Welcome to Code Wars! This is the new website for the codewars. You can find more information and previous iterations of the code wars in the tabs. The challenges can be found inside the Challenges tab.


In the Challenges page, you will find 9 challenges to complete. In this iteration, you will need to complete each challenge in sequence. You will need to access the first challenge (Cipher challenge) and when completed you will receive a link to the next challenge. Each challenge will have their own set of instructions so all the details and isntructions can be found inside the challenge's pages.


Each challenge is worth 10 points and the winner of this iteration of the code wars will be the team with most points or the team that finished all the challenges first. If your team get stuck in a challenge, you can call the organizing team and we can give you hints to complete the challenge.


Good luck and remember to have fun!!
