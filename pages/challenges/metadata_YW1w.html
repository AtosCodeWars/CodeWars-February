Metadata Challenge

Welcome to the Metadata Challenge. Below, you will find a image that has some hidden messages in it. Your task is to find the message and write it down on the input below (please use all lowercase letters when entering the phrase). There are two hints you will need to find.

1. There are some hidden messages inside the colored boxes of the image. You will need to play with the contrast of the image. We recommend you use this website to modify the image: https://www5.lunapic.com/editor/

2. You will need look at the metadata of the image to find out which is the correct text you need to enter in the input below.
